Power Distribution Network (PDN)

Complete PDN design guide. Voltage regulation, decoupling capacitors, power straps, IR drop analysis, simulation, and production optimization.

By EcrioniX · Published June 14, 2026 · ~4600 words · 14 min read

1. PDN Architecture

PDN delivers stable power from package to every gate on the chip. Poor PDN causes voltage droop, timing violations, and functional failures.

PDN Hierarchy (Power Path): Package (PCB Level): ┌─────────────────────────────┐ │ Bulk Capacitors (100µF) │ Lowest impedance at low frequency │ Ceramic Caps (10µF-100µF) │ Mid-frequency response └─────────────────────────────┘ ↓ (Package BGA Balls) Die (Chip Level): ┌─────────────────────────────┐ │ On-Die Decoupling Caps (1µF-10µF) │ High-frequency response │ Integrated Voltage Regulators │ Fast local regulation └─────────────────────────────┘ ↓ (Power Metal Straps) Cell Level: ┌─────────────────────────────┐ │ Local Power Rails (0.1-1 µm width) │ Ultra-high frequency │ Distributed to every cell │ Minimal loop inductance └─────────────────────────────┘ Impedance vs Frequency: - DC: Package impedance dominates (mΩ) - kHz: On-die caps dominate (mΩ) - MHz: Metal straps dominate (µΩ) - GHz: Local power rails dominate (nΩ)

2. IR Drop Analysis

Voltage drop = Current × Resistance (Ohm's Law)

Static IR Drop (DC): I_peak = peak current (A) R_path = resistance from pad to gate (Ω) V_drop = I_peak × R_path Dynamic IR Drop (transient): di/dt = rate of current change (A/ns) L = inductance of power path (nH) V_droop = L × di/dt Typical budget: Total allowed drop = 10% of supply voltage For 1.8V supply: max 180mV drop Static component: 100-120mV Dynamic component: 60-80mV Example (5nm chip): Peak current: 50A (all cores active) R_path: 5mΩ V_static = 50A × 5mΩ = 250mV (EXCEEDS budget!) Solution: Reduce R_path to 3.6mΩ OR add decoupling capacitors for dynamic support

3. Decoupling Capacitors

On-die capacitors provide local charge storage to support current spikes and reduce voltage droop.

Types of decaps:

Capacitor placement rule: Place near power consumers (CPU cores, memory macros). Dense clustering improves high-frequency response.

4. Power Strap Design

Power straps are wide metal lines that distribute power from pads to all cells.

Design considerations:

  • Strap width: 1-10 µm (wider = lower resistance)
  • Strap spacing: 50-200 µm (closer = lower voltage drop at cell level)
  • Strap direction: Vertical and horizontal (create grid for uniform distribution)
  • Metal layers used: Top metal layers (thicker, lower resistance)
  • Return path (ground): Equal or greater width than power (minimize loop inductance)
Power Strap Grid Layout (Top-Down View): VDD (Horizontal Straps - M9) ┌───┬───┬───┬───┬───┬───┬───┬───┐ │ │ │ │ │ │ │ │ │ ├───┼───┼───┼───┼───┼───┼───┼───┤ GND (Vertical Straps - M8) │ │ │ │ │ │ │ │ │ ├───┼───┼───┼───┼───┼───┼───┼───┤ │ │ │ │ │ │ │ │ │ ├───┼───┼───┼───┼───┼───┼───┼───┤ │ │ │ │ │ │ │ │ │ └───┴───┴───┴───┴───┴───┴───┴───┘ Spacing: Horizontal: VDD straps every 100µm Vertical: GND straps every 100µm Width: 2µm (both VDD and GND for balance) Metal layers: M9 (top) and M8 (sub-top) Resistance per unit length (7nm): M8: ~5 mΩ/mm M9: ~3 mΩ/mm

Power Strap Grid Design (7nm)

Die size: 50mm × 50mm Core area utilization: 75% Peak current: 100A distributed Target IR drop: 150mV max Power grid: - Strap width: 2 µm (M8 metal layer, high conductivity) - Strap pitch: 100 µm (spacing between straps) - Metal height: 800 nm (2.5x thickness of lower metals) - Resistance per strap: ~1 mΩ/mm Calculation: # of power straps: 50mm / 100µm = 500 straps Current per strap: 100A / 500 = 0.2A (acceptable) Drop per strap: 0.2A × R_strap (dependent on strap length) Result: Uniform voltage across die with <100mV local variation

5. Electromigration

Electromigration (EM) is metal failure caused by high current density. Electrons physically move metal atoms, creating voids and open circuits.

EM lifetime (Black's equation): Mean Time To Failure (MTTF) = A × (J^-2) × exp(Ea / kT) where: J = current density (A/cm²) A = empirical constant (depends on metal, process) Ea = activation energy (~0.5 eV for copper) k = Boltzmann constant T = absolute temperature (K) Example (5nm, copper metal): Design rule: J_max = 1 MA/cm² Strap 1µm wide, 800nm thick: Area = 1µm × 0.8µm = 0.8 µm² = 0.8 × 10^-8 cm² Max current = 1 MA/cm² × 0.8 × 10^-8 cm² = 0.8A per strap For 100A peak current: need 100A / 0.8A = 125 straps minimum Our design with 500 straps → SAFE (400% margin)

6. PDN Simulation

PDN analysis uses SPICE simulation to predict voltage droop over time.

Simulation inputs:

  • Current waveform (switching activity → current profile)
  • Power delivery network model (resistors, inductors, capacitors)
  • Voltage regulator model (VRM specifications)
  • Package and die geometry

Simulation outputs:

  • Voltage waveform at critical nodes
  • Peak voltage droop (compare to budget)
  • Frequency response (impedance vs frequency plot)
  • Current distribution across straps

Production-grade PDN simulation includes:

  • Temperature variation (EM accelerates at higher temp)
  • Process variation (metal resistance can vary ±10%)
  • Aging effects (capacitor degradation over time)

7. Real-World PDN Examples

Mobile Processor (Apple A17)

  • Supply voltage: 0.8-1.3V (dynamic voltage scaling)
  • Peak current: ~80A (all cores + GPU)
  • IR drop budget: ~80mV (6% of 1.3V supply)
  • On-die capacitance: ~50 mF (across 150mm² die)
  • PDN impedance: <2 mΩ @ 1MHz

Server Processor (AMD EPYC)

  • Supply voltage: 0.6-1.2V (aggressive scaling)
  • Peak current: ~200A per chiplet
  • Multiple voltage domains (cores, cache, uncore)
  • On-die voltage regulators (integrated LDO per core)
  • PDN impedance target: <1 mΩ @ 1MHz (tight!)

8. PDN Design Checklist

  • ✅ Set voltage budget: 10% of supply voltage
  • ✅ Allocate static vs dynamic droop: ~70% static, ~30% dynamic
  • ✅ Size power straps: Based on current, EM rules, target impedance
  • ✅ Design power grid: Uniform spacing, multiple layers, ground return path
  • ✅ Add decoupling capacitors: Distribute across die (near power consumers)
  • ✅ Calculate inductance: Minimize loop inductance (tight power-ground spacing)
  • ✅ Simulate PDN: SPICE-based transient analysis with current profile
  • ✅ Check electromigration: Verify J_max not exceeded anywhere
  • ✅ Thermal analysis: I²R losses in straps generate heat
  • ✅ Design review: Verify with power delivery expert
