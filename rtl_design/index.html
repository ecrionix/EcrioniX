01 Introduction to Verilog HDL

Verilog is a Hardware Description Language (HDL) used to model digital circuits at various levels of abstraction. It is widely used for RTL (Register Transfer Level) design in the VLSI industry.

History

Developed in 1984 by Gateway Design Automation, later standardized as IEEE 1364.

Abstraction Levels

  • Behavioral: High-level algorithmic description
  • RTL: Register Transfer Level (most common for synthesis)
  • Gate Level: Netlist of logic gates
  • Switch Level: Transistor level

Basic Structure

module example(input clk, input rst, input [7:0] d, output reg [7:0] q);
    always @(posedge clk or posedge rst) begin
        if (rst) q <= 8'b0;
        else q <= d;
    end
endmodule

Key Concepts in Verilog

Modules & Ports

Basic building block. Inputs, outputs, inouts.

Always Blocks

Combinational (@*) and Sequential (posedge/negedge)

Data Types

wire, reg, integer, real, etc.

Operators

Arithmetic, Logical, Bitwise, Reduction, Shift
