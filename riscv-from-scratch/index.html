RISC-V from Scratch

We're going to build a working RISC-V processor — together with Claude writing and optimizing every line of Verilog. From the instruction set to a single-cycle core, then a pipelined, optimized CPU you can run in your browser. Every module's code is copy- and download-ready.

▶ Start with Day 1 — The Plan

Most courses describe a CPU. This one builds one. Step by step, you'll assemble a real RV32I RISC-V processor in Verilog — register file, ALU, control unit, datapath — until it executes an actual program. Then we'll optimize it: pipeline it, add forwarding and hazard handling, CSRs, and I/O. Claude designs and writes the RTL, explains every decision, and you can copy or download each module and test it live in our online Verilog simulator. No prior CPU-design experience needed — just basic Verilog.
